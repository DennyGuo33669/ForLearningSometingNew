# 3.5.1 一元操作符

只能操作一个值的操作符叫做一元**操作符**

## 1.递增和递减

递增和递减操作符直接**借鉴**自C

**前置**:位于要操作的变量之前

```js .line-numbers
var age = 29;
++age;
```

等于:

```js .line-numbers
var age = 29;
age = age + 1;
```

**后置**:位于要操作的变量之后

```js .line-numbers
var age = 29;
--age;
```

等于:

```js .line-numbers
var age = 29;
age = age - 1;
```

执行**前置**递增和递减操作时,变量的值都是在语句被求值以前改变的(在计算机科学领域,这种情况通常被称作**副效应**)

```js .line-numbers
var age = 29;
var anotherAge = --age + 2;
console.log(age); // 28
console.log(anotherAge); // 30
```

由于**前置**递增和递减操作与执行语句的优先级**相等**,因此整个语句会从左至右被求值.

```js .line-numbers
var num1 = 2;
var num2 = 20;
var num3 = --num1 + num2; // 21
var num4 = num1 + num2; // 21
```

**后置**:位于要操作的变量之后

```js .line-numbers
var age = 29;
age++;
```

后置递增/递减操作符是包含他们的语句被求值之后才执行.

```js .line-numbers
var num1 = 2;
var num2 = 20;
var num3 = num1-- + num2; // 22
var num4 = num1 + num2; // 21
```
